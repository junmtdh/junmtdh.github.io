Markdown的基础语法

  • Markdown的基础语法

1.标题

在markdown语法中，标题可以使用两种标记：底线(中划线_、等号=)和#。

  • 使用底线的语法如下。

标题内容
=====

或

标题内容
-—-

说明：底线如果是等号(=)表示一级标题，底线如果是中划线(-)表示二级标题；底线符号的数量至少2个,这种底线语法只支持这两级标题。

  • 使用#的语法如下：

#+ 空格 + 标题内容

说明：#的个数表示了标题的等级，#后面需要加一个空格，最多支持六级标题。

markdown代码

# 一级标题  
## 二级标题  
### 三级标题  
#### 四级标题  
##### 五级标题  
###### 六级标题

2.段落与换行

如果行与行之间没有空行，则会被视为同一段落；如果有空行，则会被视为不同的段落。在段落中换行，只需要在上一行的末尾插入2个及以上的空格然后按回车即可。

演示结果如下：
没有空行：
我是第一行
我是第二行
有空行：
我是第一行

我是第二行
在段落中换行：
我是第一行，(如果想在段落中换行，在末尾插入2个或2个以上的空格然后回车即可换行)
我是第二行

3.粗体与斜体

3.粗体与斜体

斜体： 左右各1个星号(* *)来包裹 或 左右各1个下划线(_ _)来包裹。

如：这一行字
如：我是斜体字

粗体： 左右各用两个星号(**)包裹 或 左右各两个下划线(__)包裹。

粗斜体： 左右各用三个星号(*** ***)包裹 或 左右各三个下划线(___ ___)包裹。

4.分隔线

你可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格。

例如： 

***  
---    
_ _ _

5.删除线

如果段落上的文字要添加删除线，只需要在文字的两端加上两个波浪线 ~~ 即可

删除线语法：
~~被删除的字符~~
如：
被删除的字符

6.下划线

下划线可以通过 HTML 的 <u> 标签来实现：
语法：

<u>我有下划线</u>

预览：
我有下划线

7.脚注

脚注是对文本的补充说明，用[^要注明的文字]，注意其后勿加空格。
语法：

[^要注明的文字]

如：
阅读^book

8.Markdown列表

无序列表

星号、加号、减号 + 空格 +列表内容

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容

语法：

* 君不见
* 黄河之水天上来
+ hello world
- Hi~ o(￣▽￣)ブ

预览：

  • 君不见
  • 黄河之水天上来
  • hello world
  • Hi~ o(￣▽￣)ブ

有序列表

数字+英文句号+列表内容

有序列表使用数字并加上 . 号来表示，如：

  1. 奔流到海
  2. 不复回

列表嵌套

列表嵌套只需在子列表中的选项前面添加四个空格(一个Tab键)即可：

  1. 第一项
    • 第一项嵌套的第一个元素
    • 第一项嵌套的第二个元素
  2. 第二项
    • 第二项嵌套的第一个元素
    • 第二项嵌套的第二个元素
  3. 第三项
    • 列表嵌套

      区块引用

区块是可以嵌套的，一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推：

列表中使用区块

如果要在列表项目内放进区块，那么就需要在 > 前添加四个空格的缩进。

列表中使用区块实例如下：

11.Markdown代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`），例如：

printf() 函数

代码区块
代码区块使用 4 个空格或者一个制表符（Tab 键）。

实例如下：

<?php
echo 'RUNOOB';
function test(){
echo 'test';
}

也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）: 

$(document).ready(function () {
    alert('RUNOOB');
});

12.Markdown 链接

文字链接

链接使用方法如下：

[链接名称](链接地址)
或者
<链接地址>

方法：
[菜鸟教程](https://www.runoob.com)

预览结果：
这是一个链接 菜鸟教程

网址链接

将网络地址或者邮箱地址使用尖括号包裹起来会直接转换成超链接。
写法：
<https://www.runoob.com>
预览：
https://www.runoob.com

高级链接

我们可以在正文中引用链接标记，即引用定义好的变量。

在底部定义链接标记，可以理解为定义一个地址变量：

语法：

[链接文字][链接标记]
[链接标记]:链接地址

这个链接用 1 作为网址变量 [Google][1]
这个链接用 run 作为网址变量 [Runoob][run]
然后在文档的结尾为变量赋值（网址）
[1]: http://www.google.com/
[run]: http://www.runoob.com/
http://www.runoob.com/

13.转义

如果想在markdown中插入一些标记符号，但是又不想让这些符号被渲染，就可以使用\进行转义。
语法：

\特殊符号

14.Markdown图片

Markdown 图片语法格式如下：

![图片描述](图片地址)
![图片描述](图片地址 “可选标题”)

开头一个感叹号 !
接着一个方括号，里面放上图片的替代文字
接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上选择性的 ‘title’ 属性的文字。

15.markdown表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

格式如下：
| 表头 | 表头 |
| —- | —- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |

对齐方式
我们可以设置表格的对齐方式：

-: 设置内容和标题栏居右对齐。
:- 设置内容和标题栏居左对齐。
:-: 设置内容和标题栏居中对齐。
如下：

左对齐 右对齐 居中对齐

16.表情符号

使用冒号(:)包裹表情代码即可，语法如下：

:表情代码:

例如：
:smile:
:laughing:
:+1:
:-1:
:airplane:
全部表情的cheatsheet在https://www.webfx.com/tools/emoji-cheat-sheet/

17.任务列表

  • 任务列表的语法如下：
    • 未勾选
    • 已勾选
  • 语法说明：
    a.任务列表以减号(-) 加 空格开头，然后加中括号[]，中括号里面由空格或x组成。
    b.字母x可以是大写或小写，当中括号中的字符为空格时，表示未选中，为x表示选中。

代码： 

- [x] 去上课  
- [x] 认真学习  
- [ ] 打游戏

预览：

  • 去上课
  • 认真学习
  • 打游戏

18.锚点(书签)

用来标记文档的特定位置，使用锚点可以跳转到当前文档或其他文档中指定位置。

  • 锚点语法：

    [锚点描述](#锚点名)

  • 语法说明：
    中文可能有些网站不支持，锚点建议使用字母和数字，锚点名区分字母大小写，锚点名中不能有空格或其他特殊字符。
  • 添加锚点:
    <span id="锚点名">跳转到的地方</span>
